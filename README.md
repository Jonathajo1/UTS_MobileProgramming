# UTS Pemrograman Mobile - Aplikasi Manajemen Pengguna

Ini adalah aplikasi mobile yang dikembangkan sebagai bagian dari Ujian Tengah Semester (UTS) untuk mata kuliah Pemrograman Mobile. Aplikasi ini dibangun menggunakan Android Studio dan mencakup beberapa fitur seperti login, registrasi, dan halaman profil.

## Fitur

- **Halaman Login**: Memungkinkan pengguna untuk masuk dengan email dan kata sandi mereka. Jika kredensial benar, pengguna akan diarahkan ke aktivitas utama.
- **Halaman Registrasi**: Memungkinkan pengguna baru untuk membuat akun dengan memasukkan email, kata sandi, dan mengonfirmasi kata sandi mereka. Setelah berhasil mendaftar, pengguna dapat masuk.
- **Halaman Profil**: Menampilkan informasi profil pengguna, termasuk foto profil, nama, email, dan bio. Pengguna dapat mengklik tombol untuk membuka situs web tertentu.

## Teknologi yang Digunakan

- **Android Studio**: IDE yang digunakan untuk mengembangkan aplikasi.
- **Java**: Bahasa pemrograman yang digunakan untuk logika aplikasi.
- **ConstraintLayout**: Digunakan untuk membuat tata letak yang responsif.
- **Material Design Components**: Digunakan untuk antarmuka pengguna yang modern dan bergaya.
